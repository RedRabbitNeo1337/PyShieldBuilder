# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2026-07-01

### Added
- Initial release of PyShieldBuilder
- AES-256 encryption for Python source code
- HMAC-SHA256 integrity verification
- In-memory bytecode execution
- Secure key management
- Configuration-based security policies
- Command-line interface for building protected packages
- Dynamic loader for encrypted modules
- Comprehensive logging and audit trails
- Full test coverage
- Complete documentation
- Example implementations
